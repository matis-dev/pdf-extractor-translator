## Document Standards

All documentation follows these conventions:

- **Format:** CommonMark Markdown
- **Diagrams:** Mermaid.js (renders in GitHub, VS Code, etc.)
- **Code Blocks:** Fenced with language tags
- **Headers:** ATX style (`#`, `##`, `###`)
